# ❔ JKT48Guessr
Fanmade JKT48 Guessr Bot, Dibuat dengan Discord.js.

# 🧾 Features
- [ ] Enhanced Database
- [ ] Strike System
- [x] Difficulty
- [x] Simple Leaderboard
- [x] Slash Command
- [x] Covered Face with Text
- [ ] Using a 'Friendly' Answers, like 'Erine' should be guessed as 'Catherina' not 'Erine'

# ⏲️ Installation 
- 🔗 Install it with
> ```npm install```
- 🌬️ Try to run it with
> ```npm start```
